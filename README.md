### Setup Guide
- **Backend**
  - create a env file with appropriate env varialbles
  - `cd server`
  - `pnpm i`
  - `pnpm db:migrate`
  - `pnpm start`
- **Frontend**
  - `pnpm i`
  - `pnpm run dev`
